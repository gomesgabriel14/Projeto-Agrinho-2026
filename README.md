## O projeto foi desenvolvido estritamente de acordo com os critérios técnicos exigidos pelo regulamento do concurso:
* **HTML5:** Utilizado para a marcação estrutural e semântica da página.
* **CSS3:** Responsável por toda a estilização, paleta de cores harmoniosa e layout.
* **JavaScript (Vanilla JS):** Responsável pela lógica de programação, eventos e interações.
